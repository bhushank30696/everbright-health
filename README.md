# EverBright Health — Onboarding Intelligence Dashboard

A high-fidelity, premium React Single Page Application (SPA) built with Vite and Vanilla CSS for portfolio-level operations management, onboarding SLA tracking, and interactive provider investigations.

## 🚀 Key Features

* **Dynamic Google Sheets Integration**: Real-time periodic polling (every 60s) and manual sync controls utilizing a standalone, client-side CSV parser.
* **Practice Leads Grid & List Views**: A dynamic dual-view workspace allowing leads to switch between an obsidian dark card grid and a tabular data list for provider management.
* **Detailed Investigator Workspace**: Smooth row/card selection handoff displaying KPI metrics, SLA warnings, compliance checklists, and severity-badged escalation logs.
* **Operational Command Center**: Interactive metrics, delay funnel analytics, escalation type breakdown, and comparative market tables with live column sorting.
* **Premium Brand Identity**: Styled with PT Serif and Inter typography, Everbright Health orange accent colors, warm cream light mode, and deep forest green dark mode support.
